# Для чего нужны HTML и CSS? Какова их роль в создании веб-страницы?
* HTML создаёт скелет сайта(заголовки, текст, абзацы, списки и.т.д). CSS-задаёт стиль всему скелету(размер, цвет, шрифт) 
# Что такое GitHub Pages и как этот сервис работает?
* Хостинг статических сайтов, встроенный в платформу GitHub, который автоматически публикует HTML, CSS и JavaScript файлы прямо из вашего репозитория.
# Какие теги HTML и свойства CSS вы использовали в своей работе? Дайте им краткую характеристику.
* Color и Background-color. Color - цвет текст, Background-color - цвет фона
